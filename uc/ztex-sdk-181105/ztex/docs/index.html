SDK for ZTEX FPGA Boards - Documentation

Overview

The ZTEX SDK consists in two parts:
  • The Firmware Kit which is used for building the firmware for the EZ-USB FX2 and FX3 devices
  • The host software API written in Java (full feature set) and C (core functions).
